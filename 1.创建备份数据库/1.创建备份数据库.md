### 一、创建数据库
基本命令
```sql
CREATE DATABASE 数据库名 [DEFAULT CHARACTER SET 字符集 COLLATE 字符序]...;
```
#### 示例1: 创建数据库itcast
```sql
CREATE DATABASE itcast;
```
- 查看当前账户下的所有数据库
```sql
SHOW DATABASES;
```
- 查看某一数据库的定义语句
```sql
SHOW CREATE DATABASE 数据库名;
```
#### 示例2: 查看数据库itcast的定义语句




命令行执行sql脚本
```cmd
mysql -u 用户名 -p -D 数据库名 < 文件路径及文件名.sql
```
